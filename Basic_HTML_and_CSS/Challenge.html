The < Article > Tag


The < article > tag groups together self-contained content. The content within the < article > tags should make sense on its own independent of the rest of the site.

This < summary > tag is used with the < details > tag as a heading.

The < details > tag displays additional details that can be opened and closed on demand.
It is usually used to make the site more interactive.

The < time > Tag

The < time > is used for tagging specific times, such as 5:22.

The datetime attribute makes the time machine-readable so that users can interact with it to add it to their calanders or to improve SEO.
